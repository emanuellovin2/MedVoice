MedVoice este o aplicație PWA local-first pentru generarea automată de documente medicale Word și PDF pe bază de dictare vocală și template-uri DOCX, proiectată să funcționeze instalată pe Android și iOS ca aplicație web offline, fără dependențe obligatorii de cloud, folosind faster-whisper (sau fallback local/browser) pentru speech-to-text și docxtemplater pentru generarea documentelor; utilizatorul (asistenta medicală) selectează un template DOCX, pornește dictarea vocală și introduce informații natural, iar sistemul convertește vocea în text și segmentează automat informația în câmpuri medicale structurate precum pacient, diagnostic, tratament și recomandări, iar fiecare câmp nou este considerat complet DOAR când utilizatorul spune cuvântul „punct”, care declanșează finalizarea segmentului curent și începerea unuia nou; aplicația trebuie să funcționeze complet în browser pe mobil (iOS Safari + Android Chrome), cu suport PWA instalabil, includere de manifest.json și service worker pentru funcționare offline, cache de template-uri și sesiuni, UI optimizat touch-first cu butoane mari, contrast ridicat și flux extrem de simplu (upload template, selectare, start/stop dictare, generare document), fără chatbot sau interfețe complexe; sistemul trebuie să detecteze automat placeholder-ele din DOCX și să le mapeze în câmpuri fără configurare manuală, să salveze sesiuni local (IndexedDB sau storage local), și să genereze DOCX + PDF local sau prin fallback; voice input trebuie să funcționeze pe mobil fie prin Web Speech API dacă este disponibil, fie prin upload audio, fie fallback manual; AI este opțional și folosit doar pentru structurare și validare, nu pentru conversație; totul trebuie să fie ultra simplu, rapid, offline-capabil, instalabil ca aplicație pe telefon, cu latență mică și UX optimizat pentru utilizare în clinică.
